## Estudos de SQLite

Repositório criado para documentar meus estudos em SQLite e SQL, reunindo exercícios, consultas, desafios e anotações realizadas durante os cursos da Alura.

## Objetivo

Este repositório tem como finalidade acompanhar minha evolução no aprendizado de bancos de dados relacionais utilizando SQLite, praticando desde comandos básicos até consultas mais elaboradas.

O foco é consolidar os conhecimentos por meio de exercícios práticos e servir como material de consulta para projetos futuros.
